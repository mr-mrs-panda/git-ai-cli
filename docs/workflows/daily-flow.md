## Quick Feature Workflow

1. Make changes
2. Run `git-ai auto`
3. Review and finalize PR

## Controlled Workflow

1. Run `git-ai prepare`
2. Run `git-ai branch`
3. Make or refine changes
4. Run `git-ai stage`
5. Run `git-ai commit`
6. Run `git-ai pr`

## Usage

```bash
git-ai branch
```

## Behavior

1. Analyzes staged, unstaged, and untracked changes
2. Suggests an AI-generated branch name
3. Creates and checks out the new branch
4. Keeps your working changes available

## Branch Naming

Typical style:

- `feature/...`
- `bugfix/...`
- `chore/...`
- `refactor/...`

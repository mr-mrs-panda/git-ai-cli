## Interactive Mode

Run without arguments:

```bash
git-ai
```

## Direct Commands

```bash
git-ai auto
git-ai auto -y
git-ai auto --yolo
git-ai auto --release
git-ai prepare
git-ai branch
git-ai stage
git-ai commit
git-ai commit --single
git-ai commit --grouped
git-ai pr
git-ai release
git-ai release --no-prs
git-ai cleanup
git-ai worktree <branch-name>
git-ai unwrapped
git-ai celebrate
git-ai settings
git-ai --help
git-ai --version
```

## Unwrapped

Generate a yearly repository report:

```bash
git-ai unwrapped
git-ai unwrapped --language german
```

## Celebrate

Generate a visual PR celebration page:

```bash
git-ai celebrate
git-ai celebrate --language german
```

## Language Option

Both reports support language selection (English/German).
